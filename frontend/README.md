# Frontend

This is an initial frontend version built with React Native and Expo.

## Current status

- A basic application with a home page is running successfully in Expo Go.
- Other pages will be added once the folder structure are decided.

## How to use

1. Go to the frontend folder
   ```bash
   cd frontend
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the server
   ```bash
   npx expo start
   ```

4. Preview (follow the instruction on the terminal)
   - Download "EXPO GO" on your phone and scan the QR code shown in the terminal.
   - Or open it in a simulator.
